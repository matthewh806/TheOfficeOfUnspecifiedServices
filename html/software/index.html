Software

Below you'll find some links to some software ideas / projects I've worked on that I feel are interesting or developed enough to share.

They're mostly just prototypes or demos of ideas I've had and so very far from production ready. Therefore you'll encounter bugs / issues / horrific UI everywhere

As I've worked most recently in the audio programming domain and find myself increasingly interested in sound, most of the projects are in that domain.

All of the code projects listed below can be found in this monolithic repository I made. Plus a bunch of other applications and ideas.

  • SoxMosh - Image manipulation using audio effects
  • Pulsar - A ripoff (though infinitely less expensive) of the Teenage Engineering OP-1 Tombola sequencer
  • Cache - An experiment into frequency modulated audio steganography
